# RainCMS

- 基于 Vue 3 + TypeScript + Vite 的后台管理系统
- 根据 coderwhy 老师的课程编写
- 是个人的练习项目

## 前端工程化

- eslint + prettier + husky 实现代码提交的规范化

## 主要生产依赖

- vue `^3.2.37`
- vue-router `^4.0.13`
- pinia `^2.0.20`
- axios `^0.27.2`
- element-plus `^2.2.14`
